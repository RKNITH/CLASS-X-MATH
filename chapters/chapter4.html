Chapter 4 - द्विघात समीकरण

द्विघात समीकरण

द्विघात समीकरण (Quadratic Equation) एक ऐसा बहुपद समीकरण है जिसकी डिग्री 2 होती है। इसे निम्नलिखित रूप में लिखा जा सकता है: ax2 + bx + c = 0, जहाँ 'a', 'b', और 'c' गुणांक (coefficients) हैं, और 'x' चर (variable) है।

द्विघात समीकरण का सामान्य रूप

द्विघात समीकरण का सामान्य रूप इस प्रकार है:

ax2 + bx + c = 0, जहाँ a ≠ 0

यहाँ पर,

  • a x2 का गुणांक है (द्विघात पद का)।
  • b x का गुणांक है (रैखिक पद का)।
  • c एक स्थिरांक है।

ध्यान दें: 'a' का शून्य से भिन्न होना आवश्यक है क्योंकि यदि a = 0 होता है, तो समीकरण रैखिक (linear) बन जाएगा, न कि द्विघात।

मूल क्या हैं?

द्विघात समीकरण के मूल (roots) वे मान होते हैं जिनके लिए ax2 + bx + c = 0 समीकरण संतुलित होता है। ये वे बिंदु हैं जहाँ द्विघात समीकरण का ग्राफ x-अक्ष को प्रतिच्छेदित करता है।

मूल वास्तविक या सम्मिश्र (complex) हो सकते हैं, जो Δ = b2 - 4ac (विवेक्तर) पर निर्भर करते हैं।

मूल और गुणांक के बीच संबंध

द्विघात समीकरण के मूल और गुणांक के बीच निम्नलिखित संबंध होते हैं:

  • मूलों का योग: α + β = -b/a
  • मूलों का गुणनफल: αβ = c/a

जहाँ 'α' और 'β' द्विघात समीकरण के मूल हैं।

द्विघात समीकरण के अधिकतम मूलों की संख्या

द्विघात समीकरण के अधिकतम 2 मूल हो सकते हैं। ये मूल वास्तविक या सम्मिश्र हो सकते हैं।

मूलों के प्रकार (Types of Roots)

द्विघात समीकरण के मूलों का प्रकार विवेक्तर (discriminant) Δ = b2 - 4ac के मान पर निर्भर करता है:

  • Δ > 0: यदि Δ का मान शून्य से बड़ा होता है, तो मूल वास्तविक (real) और भिन्न (distinct) होते हैं।

    • उदाहरण: x2 - 3x + 2 = 0

    यहाँ a = 1, b = -3, और c = 2 है।

    Δ = (-3)2 - 4(1)(2) = 9 - 8 = 1

    क्योंकि Δ > 0, मूल वास्तविक और भिन्न होंगे।

  • Δ = 0: यदि Δ का मान शून्य के बराबर होता है, तो मूल वास्तविक और समान (equal) होते हैं।

    • उदाहरण: x2 - 4x + 4 = 0

    यहाँ a = 1, b = -4, और c = 4 है।

    Δ = (-4)2 - 4(1)(4) = 16 - 16 = 0

    क्योंकि Δ = 0, मूल वास्तविक और समान होंगे।

  • Δ < 0: यदि Δ का मान शून्य से छोटा होता है, तो मूल सम्मिश्र (complex) होते हैं।

    • उदाहरण: x2 + 4x + 5 = 0

    यहाँ a = 1, b = 4, और c = 5 है।

    Δ = 42 - 4(1)(5) = 16 - 20 = -4

    क्योंकि Δ < 0, मूल सम्मिश्र होंगे।

द्विघात समीकरण को हल करने के विभिन्न तरीके

द्विघात समीकरण को हल करने के लिए निम्नलिखित विधियाँ प्रयोग की जाती हैं:

1. कारक विधि (Factorization Method)

इस विधि में द्विघात समीकरण को दो रैखिक समीकरणों के रूप में विभाजित किया जाता है।

उदाहरण: 2x2 - 8x + 6 = 0 को हल करने के लिए:

  1. सबसे पहले, गुणांक 'a' और 'c' का गुणनफल लेते हैं: 2 * 6 = 12
  2. अब, 'b' को दो भागों में विभाजित करते हैं जिनका गुणनफल 12 हो: -8 = -6 - 2
  3. समीकरण को विभाजित करते हैं: 2x2 - 6x - 2x + 6 = 0
  4. कारक निकालते हैं: 2x(x - 3) - 2(x - 3) = 0
  5. अंत में, (2x - 2)(x - 3) = 0 से मूल निकालते हैं: x = 1, 3

2. पूर्ण वर्ग विधि (Completing the Square Method)

इस विधि में समीकरण को पूर्ण वर्ग के रूप में बदलकर हल किया जाता है।

उदाहरण: x2 - 6x + 5 = 0 को हल करने के लिए:

  1. समीकरण के दाएँ हाथ पर स्थिरांक 'c' को हटा दें: x2 - 6x = -5
  2. अब 'x' के गुणांक का आधा लेते हैं और उसका वर्ग जोड़ते हैं: (6/2)2 = 9
  3. समीकरण में जोड़ते हैं: x2 - 6x + 9 = 4
  4. अब, पूर्ण वर्ग रूप में बदलते हैं: (x - 3)2 = 4
  5. अंत में, वर्गमूल निकालते हैं: x - 3 = ±2, और मूल मिलते हैं: x = 1, 5

3. सूत्र विधि (Quadratic Formula Method)

इस विधि में निम्नलिखित सूत्र का प्रयोग करके द्विघात समीकरण के मूल निकाले जाते हैं:

x = (-b ± √(b2 - 4ac)) / 2a

इस विधि का उपयोग विशेष रूप से तब किया जाता है जब द्विघात समीकरण को कारक विधि या पूर्ण वर्ग विधि से हल करना कठिन होता है।

उदाहरण: x2 - 4x - 5 = 0 को हल करने के लिए:

  1. यहाँ a = 1, b = -4, और c = -5 है।
  2. सूत्र में मानों को प्रतिस्थापित करें: x = [ -(-4) ± √((-4)2 - 4(1)(-5)) ] / 2(1)
  3. सरलीकरण करें: x = [ 4 ± √(16 + 20) ] / 2
  4. सरलीकरण जारी रखें: x = [ 4 ± √36 ] / 2
  5. अंतिम मूल प्राप्त करें: x = (4 ± 6) / 2
  6. मूल: x = (4 + 6) / 2 = 10 / 2 = 5 और x = (4 - 6) / 2 = -2 / 2 = -1

अतः x = 5 और x = -1 इस समीकरण के मूल हैं।

मूल दिए जाने पर द्विघात समीकरण बनाना

जब किसी द्विघात समीकरण के मूल दिए जाते हैं, तो उन मूलों के आधार पर हम उस द्विघात समीकरण का निर्माण कर सकते हैं।

मूलों का उपयोग करके द्विघात समीकरण बनाने की विधि

यदि किसी द्विघात समीकरण के मूल α (अल्फा) और β (बीटा) हैं, तो उस समीकरण का सामान्य रूप निम्नलिखित होगा:

x2 - (α + β)x + αβ = 0

चरण-दर-चरण विधि

  1. मूलों का योग (Sum of roots) निकालें: α + β
  2. मूलों का गुणनफल (Product of roots) निकालें: αβ
  3. इन मूल्यों को x2 - (α + β)x + αβ = 0 में प्रतिस्थापित करें।
  4. समीकरण को सरल करें और मानक रूप में लिखें।

उदाहरण 1: जब मूल 3 और 5 दिए हों

मूल: α = 3, β = 5

  1. मूलों का योग: α + β = 3 + 5 = 8
  2. मूलों का गुणनफल: αβ = 3 × 5 = 15
  3. समीकरण बनाएँ: x2 - 8x + 15 = 0

अतः समीकरण है: x2 - 8x + 15 = 0

उदाहरण 2: जब मूल -2 और 7/2 दिए हों

मूल: α = -2, β = 7/2

  1. मूलों का योग: α + β = -2 + 7/2 = -4/2 + 7/2 = 3/2
  2. मूलों का गुणनफल: αβ = -2 × 7/2 = -7
  3. समीकरण बनाएँ: x2 - (3/2)x - 7 = 0
  4. समीकरण को गुणा करके संख्याओं को पूर्णांक बनाएँ: 2x2 - 3x - 14 = 0

अतः समीकरण है: 2x2 - 3x - 14 = 0
